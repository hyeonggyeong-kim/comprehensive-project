# 머신러닝 기반 OBD를 활용한 차량 상태 진단 및 운전 성향 분석 서비스

> 선문대학교 컴퓨터공학과 종합프로젝트 8팀

## 프로젝트 개요

OBD 단말기를 통해 수집한 차량 센서 데이터를 기반으로,  
머신러닝 모델을 활용하여 운전 습관을 분석하고 차량 상태를 진단하는 시스템입니다.

- 블루투스 통신으로 OBD 단말기에서 실시간 데이터 수신
- ML 모델로 운전 점수(Eco Score) 및 난폭 운전 여부 분류
- Flutter 앱을 통해 운전 기록 및 피드백 제공

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | Flutter (Dart) |
| Backend | Spring Boot (Java 17), JPA |
| Database | MySQL |
| ML Model | LightGBM |
| Build Tool | Gradle |

## 최종 모델 선정

> **LightGBM** 선정 (MAE: 0.6558 / R²: 0.9824)

- Transformer가 초기 비교에서 MAE 소폭 우세했으나, 하이퍼파라미터 튜닝 후 LightGBM을 선정
- 실시간 추론 적합성 및 경량화 측면에서 LightGBM이 모바일 환경에 유리
- 교차검증 결과 안정적인 일반화 성능 확인
